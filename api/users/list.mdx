---
title: 'List Users'
api: 'GET /api/user'
description: 'Retrieve a list of all users in the system'
---

## Endpoint

```
GET /api/user
```

## Authentication

This endpoint requires authentication and admin privileges.

- **verifyToken**: Valid JWT token required
- **isAdmin**: Admin role required

## Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

Returns an array of user objects.

<ResponseField name="users" type="array">
  Array of user objects
  
  <ResponseField name="_id" type="string">
    User's unique identifier
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    User's first name
  </ResponseField>
  
  <ResponseField name="apellidoPaterno" type="string">
    User's paternal last name
  </ResponseField>
  
  <ResponseField name="apellidoMaterno" type="string">
    User's maternal last name
  </ResponseField>
  
  <ResponseField name="dni" type="string">
    User's national identification number
  </ResponseField>
  
  <ResponseField name="cumpleanios" type="string">
    User's birthday
  </ResponseField>
  
  <ResponseField name="genero" type="string">
    User's gender
  </ResponseField>
  
  <ResponseField name="img" type="string">
    User's profile image URL
  </ResponseField>
  
  <ResponseField name="celular" type="string">
    User's phone number
  </ResponseField>
  
  <ResponseField name="direccion" type="string">
    User's address
  </ResponseField>
  
  <ResponseField name="correo" type="string">
    User's email address
  </ResponseField>
  
  <ResponseField name="estado" type="string">
    User's status (activo or inactivo)
  </ResponseField>
  
  <ResponseField name="rol" type="string">
    User's role in the system
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.example.com/api/user \
  -H "Authorization: Bearer your_jwt_token_here"
```

## Error Responses

### 401 Unauthorized

Returned when the authentication token is missing or invalid.

### 403 Forbidden

Returned when the user does not have admin privileges.

### 500 Internal Server Error

```json
{
  "error": "Error interno del servidor"
}
```

---
title: 'Create User'
api: 'POST /api/user'
description: 'Create a new user in the system'
---

## Endpoint

```
POST /api/user
```

## Authentication

No authentication required.

## Request Body

<ParamField body="nombre" type="string">
  User's first name
</ParamField>

<ParamField body="apellidoPaterno" type="string">
  User's paternal last name
</ParamField>

<ParamField body="apellidoMaterno" type="string">
  User's maternal last name
</ParamField>

<ParamField body="dni" type="string" required>
  User's national identification number (DNI). Must be unique.
</ParamField>

<ParamField body="cumpleanios" type="string">
  User's birthday
</ParamField>

<ParamField body="genero" type="string">
  User's gender
</ParamField>

<ParamField body="img" type="string">
  User's profile image URL
</ParamField>

<ParamField body="celular" type="string">
  User's phone number
</ParamField>

<ParamField body="direccion" type="string">
  User's address
</ParamField>

<ParamField body="correo" type="string">
  User's email address. Must be unique.
</ParamField>

<ParamField body="password" type="string" required>
  User's password
</ParamField>

<ParamField body="estado" type="string" default="activo">
  User's status. Allowed values: `activo`, `inactivo`
</ParamField>

## Example Request

```bash

  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Juan",
    "apellidoPaterno": "Perez",
    "apellidoMaterno": "Garcia",
    "dni": "12345678",
    "cumpleanios": "1990-01-15",
    "genero": "masculino",
    "celular": "+51987654321",
    "direccion": "Av. Principal 123, Lima",
    "correo": "juan.perez@example.com",
    "password": "securePassword123",
    "estado": "activo"
  }'
```

---
title: 'Create Appointment'
api: 'POST /api/cita'
description: 'Create a new appointment for a patient with a therapist'
---

This endpoint creates a new appointment and sends a confirmation email to the administrator.

## Request Body

<ParamField body="paciente" type="string" required>
  Patient ID (MongoDB ObjectId)
</ParamField>

<ParamField body="terapeuta" type="string" required>
  Therapist ID (MongoDB ObjectId)
</ParamField>

<ParamField body="fecha" type="string" required>
  Appointment date in ISO 8601 format (e.g., "2026-03-15")
</ParamField>

<ParamField body="hora" type="string" required>
  Appointment time (e.g., "14:30")
</ParamField>

<ParamField body="motivo" type="string">
  Reason for the appointment
</ParamField>

<ParamField body="precio" type="number" default="60">
  Appointment price in Soles (S/)
</ParamField>

<ParamField body="estado" type="string" default="pendiente">
  Appointment status. Possible values: `pendiente`, `confirmada`, `cancelada`, `completada`
</ParamField>

## Response

<ResponseField name="_id" type="string">
  Unique appointment identifier
</ResponseField>

  <ResponseField name="apellidoPaterno" type="string">

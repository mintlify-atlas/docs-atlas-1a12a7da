---
title: 'Update Appointment'
api: 'PUT /api/cita/update/{id}'
description: 'Update an existing appointment'
---

This endpoint updates an appointment's information. Commonly used to change the appointment status (e.g., from "pendiente" to "confirmada").

## Path Parameters

<ParamField path="id" type="string" required>
  Appointment ID (MongoDB ObjectId)
</ParamField>

## Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="paciente" type="string">
  Patient ID (MongoDB ObjectId)
</ParamField>

<ParamField body="terapeuta" type="string">
  Therapist ID (MongoDB ObjectId)
</ParamField>

<ParamField body="fecha" type="string">
  Appointment date in ISO 8601 format
</ParamField>

<ParamField body="hora" type="string">
  Appointment time (e.g., "14:30")
</ParamField>

<ParamField body="motivo" type="string">
  Reason for the appointment
</ParamField>

<ParamField body="precio" type="number">
  Appointment price in Soles (S/)
</ParamField>

<ParamField body="estado" type="string">
  Appointment status. Possible values: `pendiente`, `confirmada`, `cancelada`, `completada`
</ParamField>

<RequestExample>

```bash cURL
curl -X PUT https://api.fisioterapia.com/api/cita/update/507f1f77bcf86cd799439013 \
  -H "Content-Type: application/json" \
  -d '{
    "estado": "confirmada"
  }'
```

```javascript JavaScript
const appointmentId = '507f1f77bcf86cd799439013';
const response = await fetch(`https://api.fisioterapia.com/api/cita/update/${appointmentId}`, {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    estado: 'confirmada'
  })
});

const data = await response.json();
```

```python Python
import requests

appointment_id = '507f1f77bcf86cd799439013'
response = requests.put(
    f'https://api.fisioterapia.com/api/cita/update/{appointment_id}',
    json={'estado': 'confirmada'}
)

data = response.json()
```

---
title: 'List Appointments'
api: 'GET /api/cita'
description: 'Retrieve all appointments in the system'
---

This endpoint returns a list of all appointments with populated patient and therapist information.

## Response

Returns an array of appointment objects.

<ResponseField name="appointments" type="array">
  Array of appointment objects
  
  <ResponseField name="_id" type="string">
    Unique appointment identifier
  </ResponseField>
  
  <ResponseField name="paciente" type="object">
    Patient information (populated)
    <ResponseField name="_id" type="string">
      Patient ID
    </ResponseField>
    <ResponseField name="nombre" type="string">
      Patient first name
    </ResponseField>
    <ResponseField name="apellidoPaterno" type="string">
      Patient paternal surname
    </ResponseField>
    <ResponseField name="apellidoMaterno" type="string">
      Patient maternal surname
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="terapeuta" type="object">
    Therapist information (populated)
    <ResponseField name="_id" type="string">
      Therapist ID
    </ResponseField>
    <ResponseField name="nombre" type="string">
      Therapist first name
    </ResponseField>
    <ResponseField name="apellidoPaterno" type="string">
      Therapist paternal surname
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="fecha" type="string">
    Appointment date
  </ResponseField>
  
  <ResponseField name="hora" type="string">
    Appointment time
  </ResponseField>
  
  <ResponseField name="motivo" type="string">
    Reason for the appointment
  </ResponseField>
  
  <ResponseField name="precio" type="number">
    Appointment price
  </ResponseField>
  
  <ResponseField name="estado" type="string">
    Appointment status (pendiente, confirmada, cancelada, completada)
  </ResponseField>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET https://api.fisioterapia.com/api/cita
```

```javascript JavaScript
const response = await fetch('https://api.fisioterapia.com/api/cita');
const data = await response.json();
```

```python Python
import requests

response = requests.get('https://api.fisioterapia.com/api/cita')
data = response.json()
```

</RequestExample>
